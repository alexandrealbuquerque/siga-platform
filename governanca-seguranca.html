# Governança e Segurança O SIGA foi projetado para operar exclusivamente dentro do tenant Microsoft da cooperativa. ## Princípios de Segurança - Segurança nativa baseada em Security Roles do Dataverse - Controle em nível de registro - Isolamento completo por tenant - Nenhum banco externo - Nenhuma infraestrutura paralela ## Governança - Núcleo estrutural protegido - Separação entre configuração e operação - Versionamento via Solution Managed - Modelo modular controlado - Permissões granulares por Perfil + Setor + Módulo + Ação ## Conformidade - Compatível com políticas de TI baseadas em Microsoft - Integração nativa com Power BI - Auditoria estruturada - Controle de acesso centralizado O SIGA respeita as políticas de governança corporativa e foi desenhado para ambientes financeiros com alto nível de exigência.
