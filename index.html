SIGA – Sistema Integrado de Gestão de Atendimentos

Visão

Plataforma corporativa extensível baseada na Microsoft Power Platform, com governança forte e controle granular de permissões.

Arquitetura

  • Dataverse como núcleo estrutural
  • Segurança nativa + controle funcional
  • Model-Driven para configuração
  • Canvas para operação
  • Modelo meta-driven híbrido

Status

Projeto em fase de estruturação arquitetural (Versão 2.0 validada).
